Getting Started

Step 1: Clone the repository

git clone https://github.com/YOUR-USERNAME/vueJs.first.git


Step 2: Install dependencies

npm install


Step 3: Start the development server

npm run dev


Open your browser and go to the local URL shown in the terminal (usually http://localhost:5173).

Features

Vue 3 + Vite setup

Reactive state using ref()

Event handling (@click)



This project is intended for learning Vue.js basics and practicing the Composition API with <script setup>. It demonstrates how to:

Handle state reactively

Create dynamic templates

Render lists and conditionally display content

Respond to user interactions
